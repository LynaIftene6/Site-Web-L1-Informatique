# Site-Web-L1-Informatique
Ce projet est un site web statique réalisé en HTML et CSS dans le cadre de ma première année de licence en informatique.

L’objectif était de découvrir les bases du développement web front-end, notamment la structuration d’une page web et la mise en forme avec CSS.

---

## Technologies utilisées

- HTML5
- CSS3

---

## Objectifs du projet

- Apprendre à structurer une page web en HTML
- Comprendre les bases du CSS (mise en page, couleurs, typographie)
- Créer un site web simple et responsive (selon le projet)
- Organiser un projet web de manière claire

---

## Contenu du projet

- `accueil.html` : page principale du site
- `style.css` : feuille de style

---

## Lancer le projet

Il suffit d’ouvrir le fichier `accueil.html` dans un navigateur web.

---

## Contexte

Projet réalisé en L1 Informatique dans le cadre de l’apprentissage du développement web.
